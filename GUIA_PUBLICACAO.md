# 📱 PericiasMed — Publicar como PWA no Android

Siga este guia uma única vez. Depois o app fica disponível para sempre no seu celular.

---

## Passo 1 — Criar conta no GitHub (grátis)

1. Acesse **github.com** e clique em "Sign up"
2. Use qualquer e-mail e crie uma senha
3. Confirme o e-mail recebido

---

## Passo 2 — Criar o repositório

1. Clique no **+** (canto superior direito) → "New repository"
2. Nome do repositório: `pericias-med`
3. Marque **Public**
4. Marque **Add a README file**
5. Clique em **Create repository**

---

## Passo 3 — Fazer upload dos arquivos

No repositório criado, clique em **Add file → Upload files** e envie os 4 arquivos:
- `index.html`
- `manifest.json`
- `sw.js`
- `icon.png` *(opcional — use qualquer imagem 512×512)*

Depois clique em **Commit changes**.

---

## Passo 4 — Ativar o GitHub Pages

1. Vá em **Settings** (menu do repositório)
2. No menu lateral, clique em **Pages**
3. Em "Branch", selecione **main** e pasta **/ (root)**
4. Clique em **Save**
5. Aguarde ~2 minutos

Sua URL será: `https://SEU_USUARIO.github.io/pericias-med`

---

## Passo 5 — Instalar no Android

1. Abra o **Chrome** no Android
2. Digite a URL acima
3. Toque no menu ⋮ (três pontos) → **"Adicionar à tela inicial"**
4. Confirme com **Adicionar**

Pronto! O ícone aparecerá na tela inicial como um app normal. ✅

---

## ⚠️ Importante sobre os dados

Os dados ficam salvos no **armazenamento local do Chrome** no seu celular.
- Não serão perdidos ao fechar o app
- São apagados se você limpar os dados do Chrome
- **Não sincronizam** entre dispositivos diferentes

Para backup, use o botão "Exportar JSON" que está no app.
